## **Quick Reference**

### **Tag Replacements**
| ‚ùå Forbidden | ‚úÖ Allowed |
|-------------|-----------|
| `<div>` | `<Container>` |
| `<span>` | `<Span>` |
| `<p>` | `<Text>` |
| `<h1-h6>` | `<H1-H6>` |
| `<section>` | `<Section>` |
| `<button>` | `<Button>` |
| `<input>` | `<Input>` |
| `<label>` | `<Label>` |

### **Import Fixes**
```tsx

// ‚úÖ Correct
```
