## 📝 **Quick Reference Commands**

### **Local Development**
```bash
# Start dev server
npm run dev

# Start Firebase emulators
npm run emulators

# Run tests
npm run test

# Build for production
npm run build
```

### **Deployment**
```bash
# Deploy to Vercel
vercel deploy

# Deploy to Firebase
firebase deploy
```
